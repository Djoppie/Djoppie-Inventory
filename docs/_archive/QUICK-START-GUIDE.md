# Djoppie Inventory - Quick Start Guide

Een snelle handleiding om aan de slag te gaan met Djoppie Inventory.

## Inloggen

1. Ga naar de applicatie URL
2. Klik op **Inloggen**
3. Log in met je Microsoft account
4. Je wordt doorgestuurd naar het Dashboard

---

## Dagelijkse Taken

### Asset Opzoeken via QR Code

1. Klik op **Scan** in het menu
2. Richt je camera op de QR code
3. De asset details worden direct getoond

### Asset Handmatig Zoeken

1. Ga naar **Assets** in het menu
2. Gebruik de zoekbalk (asset code, serienummer, of naam)
3. Klik op een asset voor details

### Asset Status Wijzigen

1. Open de asset detail pagina
2. Klik op **Bewerken**
3. Wijzig de status in het dropdown menu
4. Klik op **Opslaan**

---

## Rollout Workflow

### Nieuwe Rollout Starten

1. Ga naar **Rollouts** → **Nieuwe Sessie**
2. Vul sessie naam en datums in
3. Klik op **Opslaan**

### Dag Toevoegen

1. Open een rollout sessie
2. Klik op **Planning Toevoegen**
3. Selecteer datum en dienst
4. Klik op **Opslaan**

### Werkplek Handmatig Toevoegen

1. Klik op de **oranje persoon+ knop** op een dag card
2. Vul de gebruikersnaam in
3. Configureer apparaten (optioneel)
4. Klik op **Opslaan**

### Werkplekken Importeren uit Azure AD

1. Klik op de **blauwe cloud knop** op een dag card
2. Selecteer de sector en dienst
3. Vink de gewenste medewerkers aan
4. Configureer standaard apparatuur
5. Klik op **Importeren**

### Rollout Uitvoeren

1. Markeer werkplekken als **Gereed** wanneer voorbereid
2. Klik op **Start Uitvoering**
3. Scan QR codes om assets te registreren
4. Markeer werkplekken als **Voltooid**

---

## Fysieke Werkplekken

### Werkplek Bekijken

1. Ga naar **Fysieke Werkplekken**
2. Zoek op code of naam
3. Klik op een werkplek voor details

### Equipment Beheren

1. Open werkplek detail pagina
2. Klik op een equipment slot
3. Scan of zoek een asset
4. De asset wordt gekoppeld aan de slot

### Bezetter Toewijzen

1. Open werkplek detail pagina
2. Ga naar de bezetter sectie
3. Zoek en selecteer een medewerker
4. De werkplek is nu toegewezen

---

## Admin Taken

### Diensten Synchroniseren

1. Ga naar **Admin** → **Diensten**
2. Klik op **Synchroniseren**
3. Diensten worden geïmporteerd uit Azure AD

### Intune Devices Synchroniseren

1. Ga naar **Admin** → **Intune Sync**
2. Klik op **Synchroniseren**
3. Devices worden geïmporteerd uit Intune

### CSV Import (Diensten)

1. Ga naar **Admin** → **Diensten**
2. Klik op **Importeren**
3. Selecteer je CSV bestand
4. Controleer de preview
5. Klik op **Importeren**

---

## Tips & Tricks

### Sneltoetsen
- `Ctrl + K` - Zoeken
- `Esc` - Dialoog sluiten

### Filters
- Gebruik multi-select filters om meerdere waarden te combineren
- Klik op het filter icoon om alle opties te zien
- Klik op **Clear** om filters te wissen

### QR Codes
- Print QR codes via de asset detail pagina
- Bulk print via rollout planning
- QR codes bevatten de asset code

### Dashboard
- Bekijk dagelijkse rollouts in de widget
- Vervallende leases worden getoond
- Klik op widgets voor meer details
