## üöÄ 5-Minute Setup

### Step 1: Install Printer Drivers

1. Go to [dymo.com/support](https://www.dymo.com/support)
2. Download drivers for **LabelWriter 400**
3. Run installer and follow prompts
4. Restart computer if prompted

### Step 2: Load Labels

1. Open the top cover of your Dymo 400
2. Insert roll of 25mm x 25mm labels
3. Adjust the label guide to fit snugly (not too tight!)
4. Close the cover - printer should auto-detect label size

### Step 3: Test Printer

1. Open Dymo Label software (came with drivers)
2. Print a test label to confirm printer works
3. Verify label prints at correct size (25mm square)

---

## üìÑ Printing Your First Label

### From Asset Detail Page

1. Navigate to any asset in Djoppie Inventory
2. Click **"Print Label"** button (in QR Code section)
3. Preview dialog opens - verify asset code is correct
4. Click **"Print"** button in dialog
5. Browser print dialog appears

### In Print Dialog

1. Select **Dymo LabelWriter 400** as printer
2. Verify settings:
   - **Paper size**: Auto or Custom (25mm x 25mm)
   - **Scale**: 100% (important!)
   - **Margins**: None or Minimum
   - **Quality**: High/Best
3. Click **Print**
4. Label prints immediately

### Verify Label

1. Check QR code is clear and fully printed
2. Verify asset code text is readable
3. Test scan QR code with phone/scanner
4. Asset code should match exactly

## üîß Troubleshooting

### Problem: Label is too small or too large

**Cause**: Scale is not set to 100%

**Solution**:
1. Open print dialog
2. Look for "Scale" or "Fit to page" setting
3. Set to **100%** or **Actual size**
4. Print again

---

### Problem: QR code doesn't scan

**Cause**: Low print quality or dirty printer

**Solution**:
1. Clean printer head with alcohol wipe
2. Use high-quality thermal labels
3. Set print quality to "High" or "Best"
4. Ensure label is flat, not curled

---

### Problem: Text is cut off

**Cause**: Margins are added, or label misaligned

**Solution**:
1. Check label is properly loaded in printer
2. Adjust label guide for snug fit
3. In print dialog, set margins to "None"
4. Verify paper size is 25mm x 25mm

---

### Problem: Printer not found in dialog

**Cause**: Drivers not installed or printer not connected

**Solution**:
1. Check USB cable is plugged in securely
2. Verify printer is powered on (green light)
3. Reinstall Dymo LabelWriter drivers
4. Restart browser and try again

---

### Problem: Print dialog shows wrong paper size

**Cause**: Printer hasn't auto-detected label size

**Solution**:
1. Open/close printer cover to trigger detection
2. Print test label from Dymo software
3. In browser print dialog, select "Custom" size
4. Manually enter 25mm x 25mm

## üì± Mobile Printing

### iOS (iPad/iPhone)

Mobile printing is **not recommended** for thermal labels. The Dymo 400 requires USB connection and specific drivers that aren't available on iOS.

**Alternative**: Use the desktop application, or manually write asset codes on labels.

### Android

Similar limitations - thermal printers typically require desktop drivers.

**Workaround**: Use remote desktop to access a computer with the Dymo 400 connected.

## üí° Pro Tips

### Tip 1: Print in Batches
If you need to print many labels, open multiple asset tabs and print them one after another. The printer will queue them automatically.

### Tip 2: Test Before Bulk Printing
Always print one test label before printing a large batch. Verify QR code scans correctly and all text is visible.

### Tip 3: Store Labels Properly
Keep unused thermal labels in a cool, dry place away from direct sunlight. Heat and light can darken thermal paper over time.

### Tip 4: Clean Printer Regularly
Wipe the printer head with isopropyl alcohol every 50-100 labels to maintain print quality.

### Tip 5: Use Keyboard Shortcuts
- **Print**: `Ctrl+P` (Windows) or `Cmd+P` (Mac)
- **Close Dialog**: `Esc` key
- **Navigate**: `Tab` to move between buttons

## üéØ Label Placement Best Practices

### Where to Place Labels

**Recommended Locations**:
- ‚úÖ **Laptops**: Bottom case, near serial number sticker
- ‚úÖ **Desktops**: Front or top panel, visible when deployed
- ‚úÖ **Monitors**: Back of monitor, near base
- ‚úÖ **Peripherals**: Flat surface on underside
- ‚úÖ **Mobile devices**: On protective case (not device itself)

**Avoid**:
- ‚ùå Surfaces that get hot (near vents, power supplies)
- ‚ùå Curved surfaces (QR codes may not scan well)
- ‚ùå Frequently touched areas (labels may wear quickly)
- ‚ùå Near liquids or food areas

### Durability Tips

1. **Clean Surface First**: Wipe with alcohol to remove dust/oils
2. **Apply Firmly**: Press label down for 10 seconds
3. **Avoid Stretching**: Place flat, don't pull or stretch
4. **Protect from Wear**: Consider clear laminate over label for high-use items

## üìà Label Printing Best Practices

### Organization

1. **Print labels in batches** by location, department, or asset type
2. **Keep a log** of which assets have labels printed
3. **Store spare labels** in a labeled envelope near printer
4. **Assign label duty** to specific team members on rotation

### Quality Control

1. **Scan every label** before applying to device
2. **Replace worn labels** proactively during audits
3. **Use consistent placement** across device types
4. **Document label locations** in asset notes

### Efficiency

1. **Pre-stage devices** before printing session
2. **Print and apply immediately** to avoid mix-ups
3. **Clean printer monthly** for consistent quality
4. **Order labels in bulk** to avoid running out
