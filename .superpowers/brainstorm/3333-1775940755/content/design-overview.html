Design: Overzicht & Fase 0 - Dead Code Cleanup

Eerste sectie van het volledige herorganisatie-design

Samenvatting van het project

Volledige cleanup en herorganisatie van Djoppie Inventory: dead code verwijderen, monolithen splitsen, frontend + backend + API routes hergroeperen per feature, navigatie herstructureren, en documentatie updaten. Alles wat niet actief werkt gaat weg.

Feature-Groepen (de 7 pijlers)

1. Dashboard
KPI's, overzicht, quick actions
2. Inventory
Assets, templates, QR codes, CSV import/export
3. Workplaces
Physical workplaces, equipment locaties
4. Operations
Rollouts, requests (on/offboarding), swaps
5. Devices
Intune, licenses, installed software, monitoring
6. Reports
Hardware, workplaces, operations rapportages
7. Admin
Organization sync, locaties, asset types, toekomstige RBAC
